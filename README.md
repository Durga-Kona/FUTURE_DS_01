# FUTURE_DS_01
# 📊 Business Sales Performance Analysis

## 📌 Objective

The goal of this project is to analyze business sales data to identify key trends, top-performing products, and regional performance. The project transforms raw data into meaningful insights using data analysis and visualization tools.

---

## 🛠️ Tools & Technologies

* Python (Pandas, Matplotlib, Seaborn)
* Power BI
* Jupyter Notebook
* GitHub

---

## 📂 Dataset

* Superstore Sales Dataset
* Contains data on orders, sales, profit, categories, and regions

---

## 🧹 Data Cleaning & Preparation

Using Python:

* Handled missing values
* Converted date columns to proper datetime format
* Removed inconsistencies and anomalies
* Prepared dataset for analysis

---

## 📊 Dashboard Overview (Power BI)

The interactive dashboard provides:

* 📈 **Sales Trend Analysis**
  Monthly sales performance to identify growth patterns

* 📊 **Sales by Category**
  Comparison of product categories

* 🌍 **Sales by Region**
  Regional performance insights

* 🏆 **Top 10 Products**
  Best-selling products based on revenue

* 💰 **Total Sales KPI**
  Overall business performance indicator

---

## 💡 Key Insights

* Sales show fluctuations with peak growth in later months
* Technology category generates the highest revenue
* West region leads in overall sales performance
* A few products contribute significantly to total revenue
